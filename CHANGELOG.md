# Changelog

## v1.0.0 (2026-04-23)

### 新增
- PCB 数据提取：从嘉立创EDA提取走线、过孔、器件焊盘（含位号和器件名）
- 配置面板：自动检测电源网络，支持按器件选择 Source/Load，搜索筛选，字母排序
- 支持手动添加未自动识别的网络（从全部网络列表中选择）
- KiPIDA Solver 集成：调用 `solver.py` + `mesh.py` 进行真实稀疏矩阵 IR Drop 求解
- Mesh Resolution 插值：按用户设定精度对走线进行节点插值，提升热力图密度
- 结果展示面板：文字摘要 + 3D 热力图 + 各铜层 2D 热力图（viridis 色阶）
- 分析图像自动保存到 `kipida-service/output/` 目录
- 服务地址配置：菜单入口支持运行时修改 host:port
- 连通分量分析：自动过滤孤立子图，避免矩阵奇异导致求解失败
